### Get Started

##### 1. Clone project
```
git clone https://github.com/SupeemAFK/ThaiCodex.git
```

##### 2. Create venv
```
python -m venv venv
```

##### 3. Activate venv
```
venv\Scripts\activate
```

##### 4. Install dependencies
```
pip install -r requirements.txt
```

##### 5. Pull Ollama model
```
ollama pull llama2
```

##### 6. Pull Ollama Embedding
```
ollama pull nomic-embed-text
```

##### 7. Run main file
```
python main.py
```
